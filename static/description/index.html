ReconcileX

Automated Order Settlement & Reconciliation for Odoo

The BaaSey Settlement Engine connects your Odoo Sales Orders with the P-UP Banking-as-a-Service Platform.

  • ✅ Generate a dedicated virtual account for every order
  • ✅ Track Unsettled vs Settled orders in real-time
  • ✅ Auto-mark orders as paid once payment is received
  • ✅ Automatic invoice generation and posting
  • ✅ Enterprise-grade security with HMAC signature validation

This module is ready for Odoo Online and deployable via the Odoo Marketplace.
